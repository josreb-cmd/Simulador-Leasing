# Simulador Renda Leasing
## Descrição
Este projeto é um simulador para calcular valor de renda financiamento em Leasing
## Como usar
1. Abra o ficheiro HTML no seu navegador
2. Insira a Entrada Inicial (€), se existir
3. Insira a taxa de juro anual
4. Insira o prazo total (meses)
5. Valor Residual % sobre o valor do contrato
6. O resultado será calculado automaticamente, após clicar em "EXECUTAR SIMULAÇÃO)
## Ficheiros do projeto
- `index.html` - Página principal do simulador
